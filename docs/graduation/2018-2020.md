# GIST 학부 졸업요건 - 2018~2020학번

> 2018, 2019, 2020학번 학생에게 적용되는 졸업 요건입니다.  
> 공통 요건은 [common.md](./common.md)를 참조하세요.

---

## 📋 2018~2020학번 전용 요건

### 1. GIST 새내기 (구: 신입생 세미나)

| 과목 코드          | 과목명                     | 학점  | 수강 시기            |
| ------------------ | -------------------------- | ----- | -------------------- |
| GS1901 또는 GS9301 | 신입생 세미나 / GIST새내기 | 1학점 | 1학년 1학기 의무수강 |

> ⚠️ 2018~2020학번은 **전공탐색 과목이 필수가 아닙니다.**

---

### 2. SW 기초

| 과목 코드 | 과목명         | 학점  | 비고      |
| --------- | -------------- | ----- | --------- |
| GS1490    | SW 기초와 코딩 | 2학점 | 필수 이수 |

> ✅ 컴퓨터 프로그래밍(GS1401) 이수자는 면제됩니다.

## ⚠️ 주의사항

1. **전공탐색(UC0902)은 필수가 아닙니다** - 이 과목은 2021학번부터 필수입니다.
2. **예체능 과목 수**: 2020학번부터 2과목으로 감소했습니다.
3. **과학기술과 경제(GS1701)**: 1학점 필수 이수입니다.
4. **학사논문연구 II**: 졸업예정학기에 반드시 수강해야 합니다.
